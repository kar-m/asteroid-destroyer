# Asteroid Destroyer

A 3D asteroid simulation built with [Vite](https://vitejs.dev/) and [Three.js](https://threejs.org/).

## Prerequisites

- [Node.js](https://nodejs.org/) (includes npm)

## How to Run Locally

1. **Install dependencies:**
   Open a terminal, ensure you are in the project's root directory (`asteroid-destroyer`), and run:
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open in browser:**
   Once the server starts, it will output a local URL (typically `http://localhost:5173/`). Cmd+Click on the link or open it in your web browser to view the application.

## Building for Production

If you need to build the project for production deployment:

```bash
# Create the production build in the `dist/` directory
npm run build

# Preview the production build locally
npm run preview
```
